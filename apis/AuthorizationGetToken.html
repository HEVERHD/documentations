AuthorizationGetToken Api POST

Request

Field Type Required Description Values
grant_type string Yes Type of autorization Password
username string No Receives the configured username or Sub Customer Text string
password string No Password associated with the user or sub customer Text string

Response

Field Type Description Values
access_token string It is a text string with the authorization token Text string
token_type string Defines the type of generated token Text string (bearer)
expires_in integer($int64) Defines the token expiration time in minutes Returns an Integer
.issued string($date-time) The date the token was generated Tue, 11 Jun 2024 20:27:22 GMT
.expires string($date-time) Date the token expires Tue, 11 Jun 2024 20:47:22 GMT
